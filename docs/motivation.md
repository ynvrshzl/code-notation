## Motivation
**Where did this idea begin?**

The origins of this idea came from an in-depth analysis of modern programming, historical computation, and how other complex fields, invented languages to help human beings manage immensely difficult subjects. 

We uncovered a hidden pattern in all complex fields; every field has developed some form of a notation language. So naturally, programming should have a notation language. But this hasn't been developed yet!
