# "Notation symbolic system for code"
Part of our ongoing research with [meta-lang]()

This branch of Metalang is focused on research towards **a notation language for code.**

Experimental research on notation for code. Aiming to improve human cognitive ability, while authoring computational systems. Using a proposed notation system,, as an established medium for reducing chaos, as proven across classical fields.


## Motivation
**Where did this idea begin?**

The origins of this idea came from an in-depth analysis of modern programming, historical computation, and how other complex fields, help human beings manage incredibly difficult subjects. We uncovered a hidden pattern in all complex fields; every field has developed some form of a notation language. So naturally, programming should have a notation language. But this hasn't been developed yet!
